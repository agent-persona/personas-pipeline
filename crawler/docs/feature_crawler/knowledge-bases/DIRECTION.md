# Direction Knowledge

## Current product questions

1. Where exactly does identity end and memory begin?
2. What benchmark makes a "same person?" claim credible?
3. What evidence format makes persona outputs feel trustworthy?
4. Which wedge lands first: synthetic research, persona generation, or twin chat?
5. How do we prove groundedness without exposing sensitive raw data?

## What belongs here

- category strategy
- product framing
- proof strategy
- benchmark design
- buyer and narrative questions
- shared evaluation methods

## What does not belong here

- connector-specific API notes
- feature-only implementation details
- sprint scratch notes with no durable value

## Intake loop

When new search or research happens:

1. decide whether the insight is feature-local or cross-feature
2. write a short note at the right layer
3. update the relevant `TASKS.md` or `DECISIONS.md` if the learning changed execution
4. update `STATUS.yaml` if priorities changed
