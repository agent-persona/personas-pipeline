# Commercial AI Persona Tool Landscape Analysis

## Tools Analyzed
HubSpot Make My Persona, Delve AI, Miro AI, Waalaxy, Only-B2B, Juma, Marketing Mary, UXPressia, SparkToro

## Central Finding
**No tool makes quantified accuracy claims. None offer systematic validation.**

## Feature Matrix
| Feature | HubSpot | Delve AI | Miro | Waalaxy |
|---------|---------|----------|------|---------|
| External data | None | 40+ sources | None | None |
| Accuracy claims | None | Hedged | None | None |
| Confidence scores | No | No | No | No |
| Ground-truth comparison | No | No | No | No |
| Validation features | None | Digital Twin | Team collab | None |

## Gap Analysis
Missing across ALL tools:
1. Evaluation framework / quality metrics
2. Ground-truth comparison
3. Confidence scoring per attribute
4. Predictive validation
5. Methodology transparency
6. Staleness/drift detection
7. Bias detection
8. Coverage metrics

## The 71%/18% Statistic
71% of companies exceeding revenue targets have documented personas, but only 18% have validated them with real data. The vast majority operate on unvalidated assumptions.

## Key Takeaway for Our Experiment
The commercial landscape validates our research direction — the generation-first, validation-never approach creates a wide-open opportunity for research-grounded persona evaluation. Our experiment could produce the first standardized accuracy benchmark for persona tools.
